Procedure

Aim of the experiment : Fundamental logic operations using PLC

Our aim is to implement basic logic gate function using PLC. For our better explanation, we have taken 2 input signals i.e. 2 switches connected to the input screw terminal of input module and one output (i.e. light) connected to the output screw terminal of the output module.

Apparatus required

Power supplies (for PLC, light/load), two switches, PLC, programmer (programming device).

Steps to be followed

Step 1 : To connect the switches to the corresponding input screw terminal and light to the corresponding output screw terminal.
Step 2 : To develop ladder logic that implements the required logic function.
Step 3 : To download the logic into the PLC memory.
Step 4 : Verify its truthfulness, by changing t he switch conditions and observe the behavior of the output .
